```text

새 기준 폴더:
~/Documents/Codex_프로젝트

GitHub 저장소와 로컬 폴더명:
- https://github.com/99ksy9990-bot/family-cal.git -> 01_패밀리스케줄러

- 07_비행기_예약

3. 내가 확인하면 기존 Codex 작업 폴더를 삭제 또는 보관 처리해줘.

6. 각 GitHub 폴더가 최신 상태인지 확인해줘.

주의:
- GitHub에 없는 폴더나 개인 문서는 바로 삭제하지 말고 먼저 알려줘.
- Codex 프로젝트 폴더는 바탕화면이 아니라 ~/Documents/Codex_프로젝트 아래에서 관리해줘.
- 저장소 이름은 GitHub 이름 그대로 두고, 로컬 폴더명만 위 기준으로 맞춰줘.
```
