# 제주연수원 예약 확인 프로젝트

이 프로젝트는 MG제주연수원 객실 예약 가능 여부를 확인하고, 가능할 때 예약 진행을 도와주는 작업용입니다.

## 예약 조건

- 사이트: https://mgjejuti.kfcc.co.kr/
- 날짜: 2026년 6월 21일
- 숙박: 1박 2일
- 입실 시간: 15시
- 객실: 디럭스룸
- 확인 주기: 20분마다
- 예약 가능 조건: 6월 21일 행의 디럭스룸 칸에 잔여 숫자가 있을 때

## 현재 상태

- 로그인 세션이 유지되면 잔여객실조회 화면에서 확인 가능
- 최근 확인에서는 6월 21일 디럭스룸이 마감 상태였음
- 같은 날짜에 다른 객실 잔여가 보일 수 있으나, 목표는 디럭스룸

## 진행 원칙

- 디럭스룸 잔여 숫자가 있으면 즉시 알림
- 사용자가 명시적으로 예약 진행을 승인하면 예약 절차 진행
- 결제, 최종 확정, 개인정보 입력 등 중요한 단계에서는 다시 확인
- 예약 완료 후 자동 확인 종료
- 로그인 세션이 끊기면 사용자에게 로그인 필요 상태 알림
