## 문서 분류 기준

문서 관리는 ~/Documents를 중심으로 합니다.

- 00_받은문서: 아직 분류하지 않은 임시 문서
- 01_염경섭: 염경섭 개인 문서, 사진, 관련 자료
- 02_염초롱: 염초롱 개인 문서, 학교/교육 관련 문서, 관련 자료
- 03_포스코새마을금고: 금고, 금융교육, 신입사원 교육, 업무 자료
- 04_디자인: PPT, 피그마, 이미지 소스, 폰트, 디자인 자료, 스크린샷
- 99_기타: 위 기준에 애매한 문서

99_기타에 같은 종류가 반복해서 쌓이면 새 분류 폴더로 독립시킵니다.

## 파일명 기준

일반 문서 파일은 생성일을 기준으로 `YYYY-MM-DD_분류_내용.확장자` 형식을 사용합니다.

예시:

- 2026-05-22_염경섭_주민등록등본.pdf
- 2026-05-22_염초롱_신청서.pdf
- 2026-05-22_포스코새마을금고_예금안내.pdf
- 2026-05-22_디자인_명함시안_v1.pdf

프로젝트 내부 파일, 이미지 소스, 폰트, 코드 파일은 연결이 깨질 수 있으므로 필요할 때만 별도로 정리합니다.

## Windows PC에서 쓰는 방법

다른 Windows PC에서도 같은 폴더 구조를 만들 수 있도록 `windows/setup-folders.ps1`을 제공합니다.

Windows에서 이 저장소를 내려받은 뒤 PowerShell에서 실행합니다.

```powershell
powershell -ExecutionPolicy Bypass -File .\windows\setup-folders.ps1
```

이 작업은 없는 폴더만 만들고, 기존 파일이나 폴더는 삭제하지 않습니다.
